# Test-Bevouac-Milande

This is a react application which is displaying companies that are likely to recruit after fetching an API : 

https://api.emploi-store.fr/partenaire/labonneboite/v1/company/?latitude=48.856614&longitude=2.3522219&rome_codes=M1805&sort=score&page_size=10&page=${page}

### Dependencies
- `Axios`
- `react-infinite-scroller`
- `@material-ui/lab/Pagination`

### Installation 

To launch ReactApp

- `git clone https://github.com/milly-dev/test-bevouac.git`
- `cd react-app-test-bevouac`
- `npm install`

### `yarn start` or `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
